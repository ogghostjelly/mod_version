# Mod Version

Fabric and Forge versions and version ranges.

Parser for the extended semantic version and version ranges used by Fabric described in the specification [here](https://wiki.fabricmc.net/documentation:fabric_mod_json_spec#versionrange), and parser for Maven versions and version ranges used by Forge described in the specification [here](https://cwiki.apache.org/confluence/display/MAVENOLD/Versioning) and [here](https://maven.apache.org/enforcer/enforcer-rules/versionRanges.html).
